# Product Dashboard

This is a simple dashboard that fetches and displays products from the [EscuelaJS Fake API](https://api.escuelajs.co/api/v1/products).

## Features
- Uses Bootstrap for styling
- Displays product ID, title, price, category, and image

## Usage
1. Open `index.html` in your browser.
2. The dashboard will automatically load and display the products.

## API Reference
- [Products API Documentation](https://fakeapi.platzi.com/en/rest/products/)
